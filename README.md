# 📊 InfoScan — веб-приложение для сбора данных с измерительного прибора и выгрузки CSV для SAP

Веб-приложение на чистом PHP для подключения к измерительному прибору (размеры и весы), снятия показаний, сохранения в базу данных MySQL и формирования CSV-файлов, предназначенных для последующей загрузки в SAP. Автоматизирует учёт измерений на производстве.

## 🚀 Функционал

- Подключение к измерительному прибору через сеть.
- Снятие текущих показаний в реальном времени (или ручной ввод).
- Редактирование параметров.
- Сохранение измерений в MySQL.
- Формирование CSV-файла с отобранными измерениями для загрузки в SAP.

## 🛠 Технологии

- **Backend:** PHP 7.4+ (чистый PHP, без фреймворка)
- **Frontend:** HTML, CSS (Bootstrap), JavaScript (jQuery для AJAX)
- **База данных:** MySQL
- **Связь с прибором:** PHP serial
- **Выгрузка для SAP:** формирование CSV-файлов с разделителями
- **Структура кода:** MVC-подход без фреймворка

## 📦 Установка и запуск

### Требования
- PHP ≥ 7.4 (с расширениями: mysqli, pdo_mysql, serial, zip)
- MySQL
- Веб-сервер (Apache/Nginx)
